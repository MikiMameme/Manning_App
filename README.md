Rust と GUIライブラリ `egui` を使用して作成した、シンプルで軽量なデスクトップ勤務表ビューアです。
エクセルで作成された複雑な勤務表から「今日のスタッフ配置」を自動で抽出し、ひと目で確認できるようにします。

プログラミング学習の一環として、AIアシスタントとのペアプログラミングによって開発された成果物です。

## 主な機能
- **エクセル読み込み**: `.xlsx` 形式の勤務表ファイルをドラッグ＆ドロップ、またはファイル選択で読み込みます。
- **柔軟なフォーマット対応**:
    - **日付**: `2026/2/1`（シリアル値）、`2/1`、全角の`２月１日`など、多様な表記に対応。
    - **レイアウト**: タイトル行や空行があっても自動でスキップし、正しいデータ位置（日付と名前）を動的に特定します。
（「勤務表サンプル」を参考にしてください）  

- **シフト自動集計**:
    - 「早番」「日勤」「遅番」「夜勤」のスタッフを自動で分類して表示。
- **配置チェック**:
    - シフトに空きがある（誰も配置されていない）場合、警告メッセージを表示します。
- **スクリーンショット保存**:
    - ボタンひとつで現在の画面を画像として保存・印刷できます。

## 使い方

### 起動方法
1. 配布されている実行ファイル（`Manning_proto.exe`）を実行してください。
2. 画面右上の **「エクセル読込み」** ボタンをクリックし、勤務表エクセルファイルを選択してください。
3. 読み込みが成功すると、その日のスタッフ配置が表示されます。

### 対応する勤務表の形式
一般的な「日付が横軸」「名前が縦軸」の表組みに対応しています。
- **日付**: ヘッダー行に日付が含まれている必要があります（今日の日付を自動検索します）。
- **名前**: シフト入力欄の左側にある列を「名前」として認識します。

## 💻 動作環境

- Windows 10 / 11 が正常に動作するパソコン。
- Microsoft Excel がインストールされていなくても動作します（ファイル読み込みのみ）。
